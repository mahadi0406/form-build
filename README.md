## Installation

First clone this repository, install the dependencies, and setup your .env file.

```bash
    git clone https://github.com/mdsabbirweb/form-build.git
    cd my-project
    cp .env.example .env
    composer install 
```

Then create the necessary database.

```bash
    php artisan migrate
```
